# Security Policy

## Supported versions

The current maintained version is the latest release in this repository.

## Reporting a vulnerability

Do not open a public issue for sensitive security problems. Contact the maintainer privately if possible, or open a minimal issue without secrets, tokens, private files, screenshots with credentials, or personal data.

## Scope

This repository contains Claude Code Skills, scripts and documentation. It should not contain API keys, tokens, user data, private project files or credentials.

## User responsibility

Before publishing forks or examples, check that no private files, Claude sessions, API keys, logs or real personal data are included.
