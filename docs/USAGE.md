# Usage

## App audit flow

```text
/codebase-context-loader Primero entiende este repo.
/github-app-auditor Revisa si está listo para GitHub.
/console-error-hunter Busca errores de consola y promesas rotas.
/security-privacy-review Revisa claves, localStorage y datos sensibles.
/test-plan-generator Crea pruebas finales.
/release-finalizer Déjalo como release candidate.
```

## PWA / GitHub Pages flow

```text
/pwa-offline-auditor Revisa manifest, service worker y modo offline.
/github-pages-path-fixer Corrige rutas, assets y compatibilidad con subcarpeta.
/safari-ios-compat-auditor Revisa iPhone/Safari.
/browser-compatibility-auditor Revisa Chrome, Edge, Firefox y Safari.
```

## App local-first flow

```text
/local-first-architecture Diseña la arquitectura local/offline.
/no-dependency-auditor Comprueba que no depende de CDNs ni recursos externos.
/local-storage-data-auditor Revisa persistencia, exportación y borrado.
```

## AI app flow

```text
/ai-feature-designer Decide si conviene IA real, motor local u Ollama.
/api-key-safety-planner Diseña manejo seguro de claves API.
/ollama-local-planner Planifica integración local con Ollama si aplica.
```

## Writing flow

```text
/fiction-world-bible-builder Construye la biblia narrativa antes de escribir.
/novelist-editor-hard Revisa este capítulo con auditoría editorial dura.
/brand-voice-dark Ajusta esta sinopsis con voz oscura elegante.
/kdp-launch-checklist Revisa si el libro está listo para KDP.
```

## Prompt flow

```text
/prompt-optimizer-claude Convierte esta idea en un prompt profesional para Claude.
/claude-skill-builder Convierte este prompt repetido en una Skill reusable.
```
