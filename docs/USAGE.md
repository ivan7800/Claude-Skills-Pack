# Usage

## App audit flow

```text
/codebase-context-loader Primero entiende este repo.
/github-app-auditor Revisa si está listo para GitHub.
/systematic-debugger Corrige bugs críticos sin parches a ciegas.
/security-privacy-review Revisa claves, localStorage y datos sensibles.
/test-plan-generator Crea pruebas finales.
/release-finalizer Déjalo como release candidate.
```

## PWA / GitHub Pages flow

```text
/pwa-offline-auditor Revisa manifest, service worker y modo offline.
/github-pages-deploy-check Comprueba rutas relativas, assets y compatibilidad GitHub Pages.
/safari-ios-compat-auditor Revisa iPhone/Safari.
```

## Writing flow

```text
/novelist-editor-hard Revisa este capítulo con auditoría editorial dura.
/brand-voice-dark Ajusta esta sinopsis con voz oscura elegante.
```

## Prompt flow

```text
/prompt-optimizer-claude Convierte esta idea en un prompt profesional para Claude.
/claude-skill-builder Convierte este prompt repetido en una Skill reusable.
```
