# Web App

Esta carpeta raíz funciona como una app estática para GitHub Pages.

## Funciones clave

- Catálogo de Skills.
- Copia de comandos.
- Descarga de SKILL.md.
- Generación de ZIP sin dependencias.
- PWA offline cuando se sirve por HTTPS.

## Limitaciones reales

- No puede instalar automáticamente Skills en el sistema del usuario.
- No ejecuta Claude ni Claude Code.
- No sustituye al pack real: lo presenta, documenta y facilita su uso.
