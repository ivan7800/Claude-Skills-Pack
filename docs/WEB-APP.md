# Web App

Esta carpeta raíz funciona como una app estática para GitHub Pages.

## Funciones clave

- Catálogo de 33 Skills.
- Copia de comandos.
- Descarga de SKILL.md.
- Generación de ZIP sin dependencias.
- PWA offline reforzada cuando se sirve por HTTPS.
- Favoritas y notas locales con persistencia segura: usa localStorage cuando está disponible y memoria de sesión si el navegador lo bloquea.

## Limitaciones reales

- No puede instalar automáticamente Skills en el sistema del usuario.
- No ejecuta Claude ni Claude Code.
- No sustituye al pack real: lo presenta, documenta y facilita su uso.
- La prueba final debe hacerse en la URL pública de GitHub Pages.
