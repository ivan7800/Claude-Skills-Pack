# Checklist rápido para declarar una app lista para GitHub

- [ ] Abre en local sin errores de consola.
- [ ] El flujo principal funciona 3 veces seguidas.
- [ ] Responsive probado en móvil estrecho.
- [ ] Safari/iPhone revisado si aplica.
- [ ] No hay claves ni datos reales.
- [ ] `README.md` explica uso, privacidad y publicación.
- [ ] `LICENSE`, `.gitignore`, `CHANGELOG.md` si corresponde.
- [ ] Rutas relativas compatibles con GitHub Pages.
- [ ] Service worker versionado si es PWA.
- [ ] Botón de borrar datos si usa almacenamiento local.
