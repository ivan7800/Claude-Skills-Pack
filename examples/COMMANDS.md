# Command examples

```text
/github-app-auditor mira si esta app HTML/JS está lista para GitHub Pages
/systematic-debugger el botón genera el QR pero el móvil no lo detecta
/safari-ios-compat-auditor en Safari no carga bien y el audio no suena
/readme-github-pro crea un README serio para esta app local
/release-finalizer limpia el repo y deja una release candidate
/novelist-editor-hard puntúa este capítulo como editor profesional
/brand-voice-dark reescribe esta landing con tono oscuro, sobrio y elegante
```
