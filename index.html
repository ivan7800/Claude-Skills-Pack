Instalación

Cómo usarlo con Claude Code

  1. Descarga el ZIP del pack.
  2. Descomprímelo en tu ordenador.
  3. Instálalo como Skills personales o dentro de un repo.
  4. Abre Claude Code y usa comandos como /github-app-auditor.

Windows

./install-windows.ps1

macOS / Linux

chmod +x install-macos-linux.sh
